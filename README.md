## Development Setup
For when you want to work on this project;

1. Fork on Github
2. Create a virtual environment
```shell
> virtualenv venv
> ./venv/scripts/activate
```
3. Use `pip` to install in editable mode with development requirements
```shell
> pip install -e '.[dev]'
```
4. Run the provided test suite
```shell
> python -m pytest
```
